¿Qué es Python?

Python es un lenguaje de programación de alto nivel, interpretado y de propósito general. Fue creado por Guido van Rossum en 1991. Su filosofía enfatiza la legibilidad del código con una sintaxis limpia y expresiva.

01

Variables

Una variable es un espacio en memoria que almacena un valor. En Python no necesitas declarar el tipo: se asigna automáticamente.

📦

¿Cómo se declaran?

Usa el operador = para asignar un valor. Los nombres deben comenzar con letra o guion bajo, sin espacios.

# Declaración de variables
nombre = "Ana"
edad   = 25
altura = 1.68
activo = True

print(nombre, edad)
🔄

Asignación múltiple

Python permite asignar varios valores en una sola línea, lo que hace el código más compacto.

# Múltiples variables a la vez
x, y, z = 1, 2, 3

# Misma valor a varias variables
a = b = c = 0

print(x, y, z)  # 1 2 3

Tipos de Datos

Python maneja distintos tipos de datos de forma dinámica. Los más usados son enteros, cadenas y booleanos.

Tipo int edad = 30
Tipo float pi = 3.14
Tipo str nombre = "Ana"
Tipo bool activo = True
🔢

Enteros (int)

Números sin parte decimal. Pueden ser positivos, negativos o cero.

año   = 2024
temp  = -5
cero  = 0
type(año)  # int
🔤

Cadenas (str)

Texto encerrado entre comillas simples o dobles. Inmutables.

saludo = "Hola Mundo"
letra  = 'A'
# Concatenar
msg = "Hola " + "Ana"
✅

Booleanos (bool)

Solo dos valores posibles: True o False. Muy usados en condiciones.

es_mayor = True
es_menor = False
# Comparaciones
print(5 > 3)  # True
03

Estructuras de Control

Controlan el flujo de ejecución del programa. Incluyen condicionales para tomar decisiones y bucles para repetir acciones.

🔀

Condicionales (if / elif / else)

Ejecutan un bloque de código según si una condición es verdadera o falsa.

nota = 85

if nota >= 90:
    print("Excelente")
elif nota >= 70:
    print("Aprobado")
else:
    print("Reprobado")
🔁

Bucle for

Itera sobre una secuencia (lista, rango, cadena) un número definido de veces.

# Iterar con range
for i in range(5):
    print(f"Vuelta {i}")

# Iterar una lista
frutas = ["manzana", "pera"]
for f in frutas:
    print(f)
🔄

Bucle while

Repite un bloque mientras la condición sea verdadera. Útil cuando no se conoce el número exacto de repeticiones.

contador = 0

while contador < 5:
    print(f"Contador: {contador}")
    contador += 1

print("¡Fin del bucle!")
04

Funciones

Bloques de código reutilizables que realizan una tarea específica. Se definen con def y pueden recibir parámetros y retornar valores.

01

Función simple

Sin parámetros ni retorno — solo ejecuta acciones.

def saludar():
    print("¡Hola, mundo!")

saludar()  # ¡Hola, mundo!
02

Función con parámetros

Recibe datos de entrada para operar con ellos.

def saludar_persona(nombre):
    print(f"¡Hola, {nombre}!")

saludar_persona("Ana")  # ¡Hola, Ana!
03

Función con retorno (return)

Devuelve un resultado que puede usarse en otras partes del programa.

def sumar(a, b):
    return a + b

resultado = sumar(10, 5)
print(resultado)  # 15
04

Parámetros con valor por defecto

Si no se pasa un argumento, se usa el valor predeterminado.

def potencia(base, exp=2):
    return base ** exp

print(potencia(3))     # 9  (3²)
print(potencia(2, 8))  # 256
05

Estructuras de Datos

Python incluye estructuras poderosas para organizar colecciones de datos. Las más usadas son listas y diccionarios.

📋

Listas (list)

Colecciones ordenadas y modificables. Permiten duplicados y acceso por índice.

# Crear una lista
frutas = ["manzana", "pera", "uva"]

# Acceder por índice
print(frutas[0])   # manzana

# Agregar / eliminar
frutas.append("mango")
frutas.remove("pera")

# Recorrer la lista
for f in frutas:
    print(f)
📖

Diccionarios (dict)

Colecciones de pares clave–valor. Ideales para representar objetos con atributos.

# Crear un diccionario
persona = {
    "nombre": "Ana",
    "edad":   25,
    "activo": True
}

# Acceder por clave
print(persona["nombre"])  # Ana

# Agregar / modificar
persona["ciudad"] = "Tuxtla"

# Iterar claves y valores
for k, v in persona.items():
    print(k, "→", v)
